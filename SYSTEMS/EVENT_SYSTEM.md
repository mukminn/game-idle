## üìÖ Daily Events

### Daily Login

#### Reward Schedule
```
Day 1: 100 Crystals
Day 2: 200 Crystals
Day 3: 300 Crystals + Common Materials
Day 4: 400 Crystals + Uncommon Equipment
Day 5: 500 Crystals + Rare Materials
Day 6: 600 Crystals + Epic Equipment
Day 7: 1000 Crystals + Legendary Equipment
Reset: Weekly cycle (Monday 00:00)
```

#### Streak System
```
Consecutive Days:
- 7 days: Bonus 500 Crystals
- 14 days: Bonus 1000 Crystals + Rare Equipment
- 30 days: Bonus 2000 Crystals + Epic Equipment + Title
- 100 days: Bonus 5000 Crystals + Legendary Equipment + Exclusive Title

Missed Day:
- Streak resets to Day 1
- Can use Streak Protection (cost: 100 Crystals, once per month)
```

### Daily Quests

#### Quest Types
```
1. Kill Enemies
   - Kill 100 enemies: 50 BP
   - Kill 500 enemies: 100 BP
   - Kill 1000 enemies: 200 BP

2. Complete Stages
   - Complete 5 stages: 50 BP
   - Complete 10 stages: 100 BP
   - Complete 20 stages: 200 BP

3. Upgrade Equipment
   - Upgrade 1 equipment: 50 BP
   - Upgrade 3 equipment: 100 BP
   - Upgrade 5 equipment: 200 BP

4. Complete Voyage
   - Complete 1 voyage: 100 BP
   - Complete 2 voyages: 200 BP

5. Use Skills
   - Use skills 10 times: 50 BP
   - Use skills 50 times: 100 BP
```

#### Daily Quest Rewards
```
Per Quest: 50-200 BP
Total Daily: 300 BP + 50 Crystals (completion bonus)
Reset: Daily at server reset
```

### Double Rewards Events

#### Event Types
```
1. Double BP Weekend
   - Duration: Friday-Sunday
   - Effect: 2x BP from all sources
   - Frequency: Monthly

2. Double Drop Rate
   - Duration: 3-7 days
   - Effect: 2x equipment/material drop rate
   - Frequency: Bi-weekly

3. Double Experience
   - Duration: 3-7 days
   - Effect: 2x EXP gain
   - Frequency: Monthly
```

#### Stacking
```
Events can stack:
- Double BP + VIP Bonus = 2.5x BP (VIP 5)
- Double Drop + Event Bonus = 2.2x Drop Rate
- Maximum: 3x multiplier (hard cap)
```

## üéÆ Battle Pass (Seasonal Pass)

### Pass Structure

#### Free Track
```
Levels: 30
Progress: Daily/Weekly quests
Rewards per Level:
- BP: 100-1000
- Materials: Common-Rare
- Equipment: Common-Uncommon
- Crystals: 50-200 (some levels)

Total Free Rewards:
- ~15,000 BP
- ~500 Crystals
- Various materials
- Common-Uncommon equipment
```

#### Premium Track
```
Cost: 1000 Crystals
Levels: 30 (same progress as free)
Rewards per Level:
- Crystals: 100-500
- Materials: Rare-Epic
- Equipment: Rare-Epic
- Exclusive Skins
- Exclusive Titles

Total Premium Rewards:
- ~10,000 Crystals (value)
- Rare-Epic equipment
- Exclusive cosmetics
- Premium materials
```

#### Premium+ Track
```
Cost: 2000 Crystals
Benefits:
- Instant unlock 10 levels
- All premium rewards
- Exclusive Premium+ rewards
- Early access to content
- VIP benefits (if applicable)
```

### Battle Pass Progression

#### XP Sources
```
Daily Quests:
- Per quest: 100 XP
- Daily total: 500 XP

Weekly Quests:
- Per quest: 500 XP
- Weekly total: 2000 XP

Special Quests:
- Event quests: 200-1000 XP
- Milestone quests: 500-2000 XP
```

#### Level Requirements
```
Level 1-10: 500 XP per level
Level 11-20: 750 XP per level
Level 21-30: 1000 XP per level

Total XP Required: ~20,000 XP
Duration: 4 weeks (achievable)
```

### Battle Pass Seasons

#### Season Structure
```
Duration: 4 weeks
Theme: Seasonal theme
Rewards: Themed rewards
Reset: New season starts immediately after

Seasons:
- Season 1: Launch Season
- Season 2: Fire Season
- Season 3: Ice Season
- Season 4: Dark Season
... (continues)
```
