## üë• Friend System

### Adding Friends

#### Methods
```
1. By Username/ID
   - Search username
   - Send friend request
   - Other player accepts

2. By Invite Code
   - Share invite code
   - Other player enters code
   - Auto-friend after acceptance

3. Recent Players
   - See players from leaderboard
   - Send friend request
   - One-click add
```

#### Friend Limits
```
Free Players: Max 20 friends
VIP 1-5: Max 30 friends
VIP 6-10: Max 40 friends
VIP 11+: Max 50 friends
```

### Friend Features

#### Friend List
```
Display:
- Friend username
- Friend avatar
- Friend level
- Friend current stage
- Friend power
- Last login time
- Online status (if applicable)
```

#### Friend Actions
```
- View Profile: See detailed stats
- Send Gift: Daily gift sending
- Remove Friend: Unfriend
- Block: Block user (prevent requests)
- Compare: Compare stats side-by-side
```

#### Friend Gifts

**Daily Send:**
```
- Can send to all friends
- Cost: Free
- Reward per friend: 10 BP
- Max sends: Unlimited (all friends)
- Reset: Daily at server reset
```

**Daily Receive:**
```
- Receive from friends who sent
- Reward: 10 BP per gift
- Max receives: 50 gifts (500 BP total)
- Auto-claim: VIP 10+ (optional)
- Manual claim: Free players
```

**Gift System:**
```
Mechanic:
1. Friend sends gift
2. Gift appears in mailbox
3. Player claims gift
4. Both players get reward

Rewards:
- Sender: 10 BP (sentiment reward)
- Receiver: 10 BP (actual reward)
```

### Friend Power Ranking
```
Display:
- Rank friends by power
- See top friends
- Compare with friends
- Friend milestones
```

## üéÅ Invite System

### Invite Mechanics

#### Invite Code
```
Format: USER_XXXXXX (6 alphanumeric)
Unique: Per account
Shareable: Can share via link/code
Permanent: Never expires
```

#### Invite Process
```
1. Player gets invite code
2. Share code/link dengan friend
3. Friend registers menggunakan code
4. Friend completes tutorial (Stage 10)
5. Both players get rewards
```

### Invite Rewards

#### Inviter Rewards
```
Per Friend Invited:
- 100 Crystals (immediate)
- Max: 10 friends (1000 Crystals total)

Milestone Rewards:
- 5 friends: 500 Crystals
- 10 friends: 1000 Crystals + Exclusive Skin
- 20 friends: 2000 Crystals + Legendary Equipment
- 50 friends: 5000 Crystals + Mythic Equipment + Title
```

#### Invitee Rewards
```
New Player Bonus:
- 500 Crystals (starter bonus)
- Starter Equipment Pack
- 7-day login bonus boost
- Welcome achievement
```

### Invite Tracking
```
Statistics:
- Total invites sent
- Total invites completed
- Current milestone progress
- Rewards claimed
- Next milestone
```
